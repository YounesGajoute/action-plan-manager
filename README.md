## üöÄ Quick Start

### Prerequisites
- **Node.js 18+**
- **Python 3.11+**
- **Docker & Docker Compose**
- **PostgreSQL 15+**
- **Redis 7+**

### 1. üì• Download & Setup

```bash
# Clone the repository
git clone <repository-url>
cd action-plan-manager

# Run automated setup
chmod +x setup.sh
./setup.sh
```

### 2. ‚öôÔ∏è Configuration

Edit `.env` file with your settings:

```env
# Microsoft 365 (required)
MS_TENANT_ID=your-azure-tenant-id
MS_CLIENT_ID=your-azure-client-id
MS_CLIENT_SECRET=your-azure-client-secret

# Database
DATABASE_URL=postgresql://actionplan:password@localhost:5432/actionplan

# Security
JWT_SECRET=your-very-secure-jwt-secret

# Optional: Telegram Bot
TELEGRAM_BOT_TOKEN=your-telegram-bot-token
```

### 3. üöÄ Start the Application

**Development Mode:**
```bash
./dev-start.sh
```

**Production Mode (Docker):**
```bash
./prod-start.sh
```

### 4. üìä Import Your Data

```bash

cp "Plan d'action.xlsx" ./data/

# Import the data
python scripts/import_excel.py ./data/Plan_daction.xlsx
```

### 5. üéâ Access the Application

- **Frontend**: http://localhost:3000
- **API**: http://localhost:5000
- **Grafana**: http://localhost:3001

## üö® Troubleshooting

#### Frontend Build Errors

